# Week 1 – Data Exploration and Cleaning Using Pandas

## Objective
To learn Python basics and perform basic data exploration and data cleaning using the Pandas library.

## Dataset
The dataset used for this assignment is a shopping dataset obtained from Kaggle.

## Tasks Performed
- Loaded the CSV dataset into a Pandas DataFrame.
- Explored the dataset using `head()`, `tail()`, `shape`, `columns`, and `info()`.
- Identified missing values and handled them appropriately.
- Performed basic operations such as filtering rows and selecting columns.
- Checked for and removed duplicate records.
- Created a derived column using numeric columns.
- Saved the cleaned dataset as a new CSV file.

## Files Included
- `notebook/Analysis.ipynb`
- `data/combined_dataset.csv`
- `data/cleaned_dataset.csv`
- `README.md`

## Tools Used
- Python
- Pandas

- VS Code

## Output
A cleaned dataset ready for further analysis and a Jupyter Notebook explaining each step.
